# 📖 "Ministry of Communications and Digital Economy" Module 1 Capstone project <a name="about-project"></a>

> This project is a website for ministry of communication and digital economy with 2 pages (home,about)

**[Capstone project of Microverse]** is a great project for beginners to lean how to build a simple layout

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- **HTML**
- **CSS**
- **JS**
- **Bootstrap**

<!-- Features -->

### Key Features <a name="key-features"></a>

- **[Walk through Video](https://www.loom.com/share/0e0c1485271e4613afc2fb1162336873)**
- **Error Free**
- **Responsive and User Freindly**
- **Mobile First Design**
- **Layout with Flexbox and Grid**
- **Using Js for dynamic parts and DOM manipulation**
